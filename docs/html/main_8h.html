Detailed Description

Заголовочный файл основной программы

Этот файл содержит объявление главной функции main(), которая является точкой входа в программу. В функции main() выполняется несколько ключевых операций:

  • Открытие файлов, содержащих матрицы.
  • Вычисление результата выражения с использованием функций работы с матрицами, определенных в проекте.
  • Вывод результата в консоль.
  • Освобождение выделенной памяти.

Этот файл играет важную роль в запуске основного алгоритма программы.

Объявление главной функции main()
Проверяем в функции открылись ли файлы в которых лежат матрицы, а затем решаем исходное выражение, через описанные в данном проекте функции.
В конце выводим результат в консоль и чистим память.

Эта функция инициализирует систему CUnit, добавляет тесты для разных компонентов программы, запускает их и выводит результаты. В случае ошибок в тестах выводится сообщение с кодом ошибки.

Функция выполняет следующие действия:

  1. Инициализирует реестр CUnit с помощью CU_initialize_registry().
  2. Добавляет наборы тестов для матриц, вывода и основного кода с помощью CU_add_suite().
  3. Запускает тесты для каждой категории:
    • Тесты матриц
    • Тесты вывода
    • Тесты main.c
  4. Выводит ошибки, если они возникают при выполнении тестов.
  5. Настроены различные уровни вывода:
    • Подробный вывод (CU_BRM_VERBOSE)
    • Стандартный вывод (CU_BRM_NORMAL)
    • Минимальный вывод (CU_BRM_SILENT)
  6. Запускает все тесты с помощью CU_basic_run_tests() и выводит их результаты.
  7. Очищает реестр и выводит итоговый статус.
Returns
Возвращает 0, если все тесты прошли успешно, иначе возвращает 1.
