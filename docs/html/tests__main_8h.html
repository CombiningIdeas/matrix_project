Detailed Description

Заголовочный файл для тестирования главной программы.

В этом файле содержатся объявления функций, предназначенных для тестирования основных компонентов программы, включая тесты на успешную загрузку матриц и обработку ошибок при отсутствии необходимых файлов.

Этот файл используется в сочетании с CUnit для выполнения юнит-тестирования.

int main_tests_main ( )

Главная функция для запуска тестов.

Эта функция инициализирует и запускает тесты, определенные для основного компонента программы. Она добавляет тесты в тестовый набор и сообщает о результате выполнения тестов.

Returns
0 в случае успешного выполнения тестов, в противном случае возвращает код ошибки.
