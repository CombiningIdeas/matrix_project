Detailed Description

Заголовочный файл для тестирования функций вывода матриц.

Этот файл содержит объявления функций, предназначенных для тестирования функций вывода матриц в консоль и сохранения их в файлы. Тестируется корректность работы с матрицами разных размеров, включая пустые матрицы и матрицы с одним элементом.

Тесты выполняются с использованием стандартных функций для работы с файлами и консолью.
