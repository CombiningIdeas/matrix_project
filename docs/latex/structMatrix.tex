\doxysubsection{Detailed Description}
Структура, представляющая матрицу 

Эта структура используется для представления матрицы, включающей данные в виде двумерного массива. Она содержит информацию о размерах матрицы (количестве строк и столбцов) и указатель на массив данных, который хранит элементы матрицы.

Поля\+:
\begin{DoxyItemize}
\item {\ttfamily rows} — количество строк в матрице.
\item {\ttfamily cols} — количество столбцов в матрице.
\item {\ttfamily data} — указатель на двумерный массив, содержащий элементы матрицы.
\end{DoxyItemize}

Тип данных {\ttfamily MATRIX\+\_\+\+TYPE} может быть определён как любой числовой тип, например, {\ttfamily float}, {\ttfamily double} или другой.
