\doxysubsection{Detailed Description}
Заголовочный файл для функций вывода и сохранения матрицы 

Этот файл объявляет функции для вывода матрицы в консоль и её сохранения в файл. Он предоставляет интерфейсы для отладки и сохранения результатов работы с матрицами.

Включает функции\+:
\begin{DoxyItemize}
\item {\ttfamily print\+\_\+matrix} — выводит матрицу в консоль.
\item {\ttfamily save\+\_\+matrix\+\_\+to\+\_\+file} — сохраняет матрицу в текстовый файл.
\end{DoxyItemize}
